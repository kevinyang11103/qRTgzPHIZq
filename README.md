# 漫画推荐系统 python300

## 项目概述

本项目是一款基于Python语言的漫画推荐系统，主要包含信息分析与爬虫模块。该系统通过分析用户行为与偏好，为用户推荐合适的漫画内容。

## 技术栈

### 后端技术

- Python
- Django框架

### 前端技术

- jQuery
- Bootstrap
- HTML
- JSP

### 数据库

- MySQL 5.7.33

### 开发工具

- PyCharm

## 功能模块

- 用户管理：注册、登录、个人信息管理
- 漫画推荐：根据用户行为推荐漫画
- 漫画信息管理：爬虫抓取、信息存储、更新与删除
- 漫画搜索：支持关键词搜索漫画

## 系统角色

- 普通用户：浏览漫画、搜索漫画、获取推荐
- 管理员：系统管理、用户管理、漫画信息管理

## 运行环境

- Python 3.7+
- MySQL 5.7.33
- 支持主流浏览器

## 部署步骤

1. 安装Python 3.7+
2. 安装MySQL 5.7.33
3. 克隆项目代码到本地
4. 安装依赖库（根据requirements.txt）
5. 配置数据库连接信息
6. 执行数据初始化脚本
7. 运行项目
